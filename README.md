# 📒 Notebook Project

A full-stack notebook application to create, manage, and store notes efficiently.

## 🚀 Features

* Create, edit, and delete notes
* Organized note management
* Frontend + Backend integration

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript (React)
* Backend: Node.js, Express
* Database: MongoDB

## ⚙️ Run Locally

```bash
git clone https://github.com/ankitamehla/Notebook_project.git
cd Notebook_project
npm install
cd client && npm install
cd ../server && npm install
```

Run:

```bash
cd server && npm start
cd client && npm start
```
